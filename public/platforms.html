Platform Integrations

VaultLister 3.0 connects to 9 marketplaces. Each platform uses either an official API or a browser automation bot depending on what the platform exposes.

Production ready — live with official API
Automation — Playwright browser bot
Pending — awaiting approval or credential setup

Official API

eBay

Production ready OAuth 2.0

Full listing creation, inventory sync, offer management, and order import via the eBay Sell and Buy APIs. Scopes: sell.inventory, sell.fulfillment, sell.marketing.

Setup: add EBAY_CLIENT_ID and EBAY_CLIENT_SECRET to .env, then connect via Settings → Platforms.

Etsy

Pending app approval OAuth 2.0

Listing creation, inventory management, and order sync via Etsy Open API v3. Integration is code-complete; activation is gated on Etsy developer app approval for production access.

Setup: add ETSY_API_KEY to .env once the app is approved.

Shopify

Production ready REST API

Product and inventory sync, order import, and fulfillment updates via the Shopify Admin REST API. Requires a private app or custom app token scoped to write_products and read_orders.

Setup: add SHOPIFY_STORE_DOMAIN and SHOPIFY_ACCESS_TOKEN to .env.

Depop

Production ready OAuth 2.0

Official API integration via Depop Partner API with OAuth 2.0 PKCE (S256). Listing creation, updates, deletion, and mark-as-sold with full token refresh flow.

Setup: add DEPOP_CLIENT_ID and DEPOP_CLIENT_SECRET to .env, then connect via Settings → Platforms.

Browser Automation

Poshmark

Automation Playwright

Listing creation, closet sharing, follow-back automation, and offer management via headless Chromium. Supports scheduled sharing with configurable rate limits.

Setup: add POSHMARK_EMAIL and POSHMARK_PASSWORD to .env.

Mercari

Automation Playwright

Listing creation and price updates via headless Chromium. Automatic relisting to keep items surfaced in search. Offer acceptance rules configurable per item.

Setup: add MERCARI_EMAIL and MERCARI_PASSWORD to .env.

Grailed

Automation Playwright

Listing creation and bump automation via headless Chromium. Category and size mapping handled automatically from InventoryItem attributes.

Setup: add GRAILED_EMAIL and GRAILED_PASSWORD to .env.

Facebook Marketplace

Automation Playwright

Listing creation via headless Chromium. No official seller API is available; all actions are performed through the standard web interface with human-pace rate limiting.

Setup: add FACEBOOK_EMAIL and FACEBOOK_PASSWORD to .env.

Whatnot

Automation Playwright

Inventory sync and listing creation for Whatnot shows via headless Chromium. Scheduled show reminders and lot creation from inventory items.

Setup: add WHATNOT_EMAIL and WHATNOT_PASSWORD to .env.
