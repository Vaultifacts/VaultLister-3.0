# SCOPE CONTRACT

Mission:
Ship VaultLister as a stable, deployable system.

In Scope:
- Reproducible local environment
- Deployment pipeline
- Backup + restore
- Monitoring + health checks
- Performance stability
- Documentation

Out of Scope until completion gates pass:
- Major new features
- New integrations
- Large refactors

Definition of Done:
All completion gates PASS.

Rule:
If a task does not unblock a completion gate, it goes to the backlog.
