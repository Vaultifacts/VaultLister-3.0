# VaultLister Extension Icons

This directory should contain the following icon files:

- **icon16.png** - 16x16 pixels - Used in the browser toolbar
- **icon48.png** - 48x48 pixels - Used in the extensions management page
- **icon128.png** - 128x128 pixels - Used during installation and in the Chrome Web Store

## Creating Icons

All icons should feature the VaultLister logo with:
- Purple gradient background (#6366f1 to #8b5cf6)
- White vault/box symbol
- Clean, modern design

You can create these icons using:
1. **Figma** - Design the logo at 128x128 and export at different sizes
2. **Canva** - Use the logo maker with the dimensions above
3. **Adobe Illustrator** - Create SVG and export to PNG at various sizes
4. **Online Tools** - Use favicon generators that support custom sizes

## Temporary Workaround

During development, you can use any 16x16, 48x48, and 128x128 PNG images as placeholders.

The extension will work without icons, but Chrome will show a default icon instead.
